### Установка git на ваш ПК
[Git](https://git-scm.com) скачивается с официального сайта под Вашу операционную систему.
Для пользователя Windows необходимо скачать .exe файл и при установке действуем по инструкции инстолятора.
Для пользователей Linux необходимо открыть терминал и прописать следующие данныен: *sudo apt-get install git*

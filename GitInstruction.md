## Создание и базовая работа с локальным репозиторием.
Для того чтобы создать репозиторий необходимо выбрать каталог(Папка на ПК), далее добавляем командой git init создается скрытая папка .git и инициализация вашего каталога.
Далее добавляем файл Readme.md и добавляем в отслеживаемые git add имя вашего файла, далее делаем коммит командой git commit -m "Название коммита"
